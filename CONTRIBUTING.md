# Contributing to QEVRYN
Thank you for your interest in contributing to QEVRYN, the premium local-first AI intelligence layer for Windows and Linux desktops.

## 1. Code of Conduct
We are committed to fostering an inclusive, welcoming, and safe community. Please read and adhere to our `CODE_OF_CONDUCT.md` when interacting with other contributors.

## 2. Setting Up Your Development Environment
QEVRYN is structured as a full-stack Node.js application built with Vite and React.
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the local server:
   ```bash
   npm run dev
   ```
4. Verify your build:
   ```bash
   npm run build
   ```

## 3. Contribution Guidelines
* **Privacy First**: Any feature you add must preserve user privacy. Features that silently transmit data to external servers or index directories without manual permission will be rejected instantly.
* **Branding**: Maintain the branding "Built by Veeomdecoders".
* **Tests**: All new features or routes must be accompanied by relevant test specifications.
